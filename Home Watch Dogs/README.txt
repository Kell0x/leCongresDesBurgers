PREREQUIS

Installer "Desktop visualizer", "nova launcher lite" et "UCCW"


ETAPE 1 :

Utiliser Nova et appliquer le fond d'ecran


ETAPE 2 :

Mettre le Dossier fonts dans la racine du smartphone et placer le fichier "hacked2.uzip" 
dans un dossier de votre smartphone que vous pourrez retrouver facilememt 


ETAPE 3 :

ajouter un widget UCCW et une barre de recherche google sur le haut de l'ecran
pour le UCCW clicker sur le widget puis sur le "+" et importation uzip skin, retrouvez le fichier "hacked2" 
que vous aviez mis sur votre smartphone


ETAPE 4:

ajouter un widget Desktop visualizer pour chacune des trois icones


ETAPE 5:

Selectionner l'icone -> autre -> aucun
Selectionner ACTION -> Lancer l'application -> Gmail ou Contact
OU Action -> autre -> action Nova -> Drawer pour le tiroir d'application
Etiquette : aucune
Effleurer effet => aucun
Vibration : au choix

ETAPE 6:
Adapter les paramettres nova pour que les icone de fond d'ecran soient bien associe aux widgets Desktop visualizer
